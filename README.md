# GeolocationAPI using Adonis Js

### Install Project
    $ git clone https://github.com/khobbie/GeolocationAPI.git
    $ cd GeolocationAPI
    $ npm install
    $ node ace serve --watch
    

##### Rename 'env.example' to '.env' 

Application will be running on : http://127.0.0.1:3333
